# v0.1.0 发布说明

## 主要功能

- 自动检测本机 Maven 版本、Maven Home、`mvn` 路径和 Java 版本
- 支持手动选择 Maven Home 或 `mvn` 可执行文件
- 支持多套 `settings.xml` 配置管理
- 支持新建、导入、重命名、复制、删除配置
- 支持一键设为默认 `~/.m2/settings.xml`，覆盖前自动备份
- 支持默认配置备份列表和恢复
- 支持镜像、代理、servers、pluginGroups、profiles 可视化编辑
- 支持 XML 源码编辑，并保留暂未可视化的高级 XML 片段
- 支持离线 `help:effective-settings` 试运行校验
- 提供 macOS 和 Windows 客户端打包产物

## 下载文件

- macOS: `Maven Settings Management_0.1.0_aarch64.dmg`
- Windows: `Maven Settings Management_0.1.0_x64-setup.exe`
- Windows 便携版: `mvn-settings-management.exe`

## 注意事项

- 当前版本未做正式代码签名/公证，首次运行时 macOS 或 Windows 可能出现安全提示。
- Windows 安装包使用本仓库备用 NSIS 脚本生成，安装到当前用户目录。
